# SelfStudy
**這是一款可以方便的管理單字的APP**

* SelfStudy可以輕鬆的建立您自己的英文題庫
* 可在SelfStudy中搜尋英文字典,  並將搜尋結果加入題庫 
* 多種瀏覽單字的模式
* 可備份與還原

實作方式
* 語言: Kotlin
* 相關套件: Jetpack
