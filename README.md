# SelfStudy
**這是一款可以方便的管理單字的APP**

* SelfStudy可以輕鬆的建立您自己的英文題庫
* 可在SelfStudy中搜尋英文字典, 將您搜尋的單字加入題庫 
* 多種瀏覽單字的模式
* 可備份與還原
* 白天主題和夜間主題

實作方式
* 語言: Kotlin
* 相關套件: Jetpack, Material Components
* Android架構: 單一Activity(採用Navigation), Room+LiveData+ViewModel, databindnig
* Firebase: 使用者資料之備份與還原, 下載預建立題庫
